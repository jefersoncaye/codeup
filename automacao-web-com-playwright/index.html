    Automação Web com Playwright, Python e pytest¶

    ★ 4.8 — Automação web completa do básico ao avançado. Page Object Model, fixtures, paralelismo, relatórios e CI/CD com Jenkins.

    O que você vai aprender¶

    • Instalação e configuração do Playwright com Python e pytest
    • Seletores robustos: get_by_role, get_by_text, get_by_label, locator
    • Ações: click, fill, check, select_option, hover, press, expect
    • Page Object Model — estrutura profissional de projeto
    • Fixtures pytest: setup, teardown, escopos e reutilização
    • Upload e download de arquivos, iFrames, múltiplas janelas
    • Execução paralela, relatórios HTML e comandos CLI
    • Autenticação com storage state
    • CI/CD com Jenkins

    Para quem é este curso¶

    • QAs que já sabem Python básico e querem automação web profissional
    • Desenvolvedores que querem adicionar testes E2E ao seu repertório
    • Times que precisam de uma suíte de testes Playwright bem estruturada

    Tecnologias¶

    Playwright Python pytest CI/CD
