    CodeUp — Conversion Home Page Redesign¶

    For agentic workers: REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (- [ ]) syntax for tracking.

    Goal: Transform the MkDocs home page into a conversion-focused course sales page using a Jinja2 template override, restructure course detail pages at full-name URLs, and update site navigation.

    Architecture: overrides/home.html Jinja2 template replaces the home page content block with a hero section, 5 course cards, and a consultoria form. Five new course detail directories replace the old pillar structure. Nav is restructured around courses only; tabs are removed.

    Tech Stack: MkDocs Material 9.7.6, Jinja2, CSS custom properties, GitHub Actions
