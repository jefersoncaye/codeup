Perguntas Frequentes

Os cursos são para iniciantes?

Os cursos atendem tanto iniciantes quanto profissionais que querem evoluir. Cada curso indica o nível recomendado, mas a maioria começa do zero e avança até tópicos avançados.

Preciso saber programar para começar?

Não. O curso de Python para QAs cobre tudo desde o início. Para os outros cursos, ter a base de Python ajuda mas não é obrigatório.

O acesso ao curso expira?

Não. Todos os cursos são na Udemy com acesso vitalício, incluindo atualizações futuras.

Tem suporte para dúvidas?

Sim. Você pode tirar dúvidas diretamente pela plataforma da Udemy ou usar o formulário de suporte técnico desta página para questões mais específicas.

Qual curso fazer primeiro?

Se você não tem base em Python, comece pelo Python para QAs. Se já programa, pode ir direto para Playwright Web ou API, dependendo do seu foco.
